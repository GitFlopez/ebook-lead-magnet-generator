# Ebook & Lead Magnet Generator

> Build complete lead magnets in hours, not weeks. Four prompts cover everything from content architecture to landing page copy to the launch email sequence.

## The Problem

Creating a quality lead magnet takes 20-40 hours:
- Brainstorm topic and structure → 2 hours
- Write 8-12 chapters → 15-25 hours
- Design landing page copy → 3 hours
- Write launch emails → 2-4 hours

Most people skip the lead magnet entirely, or produce something too thin to convert.

## The Solution

Four battle-tested prompts that output professional lead magnet content:

1. **Outline Generator** — Full ebook architecture in 10 minutes (chapter titles, summaries, word counts, transformation framework)
2. **Chapter Writer** — One complete chapter per run (hook, sections, examples, exercises, summary + teaser)
3. **Landing Page Copy** — Complete opt-in page (5 headline variants, benefit bullets, form copy, content upgrade variant)
4. **Launch Email Sequence** — 5 promo emails + 3 nurture emails for new subscribers

## Quick Start

```
1. Open prompts/01-outline-generator.md
2. Fill in: [TOPIC], [TARGET_READER], [DESIRED_OUTCOME], [FORMAT]
3. Run in Claude → get your complete ebook architecture
4. Repeat with prompt 2 for each chapter
5. Use prompts 3 & 4 to set up your landing page and email campaign
```

## Output Example

Input: "AI tools for freelancers" → target reader: "freelance designers & developers" → desired outcome: "book 2x clients without working more hours"

Output includes:
- Title: "The 10-Hour Freelance Week: How AI Tools Help You Earn More While Working Less"
- 10 chapters with summaries + key takeaways
- Chapter 1 full text (~2,500 words)
- Complete landing page copy
- 8-email sequence (5 promo + 3 nurture)
